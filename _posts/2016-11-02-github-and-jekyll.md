### installation

```
sudo pacman -S ruby
gem install jekyll
gem install bundler
gem install github-pages
```

### checkout source

```
git clone https://github.com/jkoz/jkoz.github.io
```

### serve

```
cd ~/jkoz.github.io && bundle exec jekyll serve
```
